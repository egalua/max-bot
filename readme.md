# Пример отправки сообщений от бота MAX в группу и пользователю

## требования
- нужно добавить бот в группу
- нужен чат пользователя с ботом
- нужны ID пользователя и ID группы
- нужены ID бота и токен бота

# запуск
- клонирование репозитория
- установка зависимостей: npm install
- добавить в корень файл .env
- сборка: npm run build
- запуск: npm run start
